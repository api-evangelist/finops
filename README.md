# FinOps (finops)
FinOps is a cloud financial management discipline combining technology, finance, and business practices to drive accountability and optimize cloud spending through budget-aware execution and cost telemetry at runtime. This index aggregates the network of FinOps platforms, standards bodies, and tooling tracked by API Evangelist.
